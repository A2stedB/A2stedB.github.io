Apple Watch是Apple的手表产品线，其中可以分为以下几个产品：
- Series数字版
- SE
- Ultra

总体来说，Apple Watch都配备了比较全的运动功能跟健康追踪功能，这几个产品之间只区别于个别的功能，大小。

# Series数字版

Series数字版的尺寸有两个：46毫米跟42毫米。可选的颜色有四个：玫瑰金色，银色，深空灰色跟亮黑色。有两种材质可以选：铝合金跟钛金属。

GPS版的价格从449欧元起，GPS加蜂窝网络版从479欧元起。钛金属绑定GPS加蜂窝网络，从799欧元起。

另外，Apple Watch有在和爱马仕合作，这系列有着独特的表带，价格1399欧元起。

# SE

SE是Apple Watch的小尺寸手表，有两个尺寸：40毫米跟44毫米。有两个颜色可以选择：午夜色跟星光色

GPS版的价格从269欧元起，GPS加蜂窝网络版从319欧元起。

# Ultra

Ultra是Apple Watch的大尺寸手表，是代表着运动手表的系列。

Ultra只有一个尺寸：49毫米。有两个颜色可以选择：原色跟黑色。只有GPS加蜂窝网络版。
价格从899欧元起。

Ultra也有爱马仕版本可以选择。

# 选择

首先，全部的手表功能都差不多，从运动方面来讲Ultra的功能是最全的，其次是Series数字版跟SE版。从健康追踪功能来说，Ultra跟Series数字版是最全的，其次是SE版。但是总体来说功能都差不多。

在这种情况下，我认为用户首先要考虑的是适合自己的尺寸，因为这是长期戴在手上的物品，与其纠结功能的差异，我认为带的舒服是首先，其次才是功能。但是不排除特别的需求。

硬件方面，目前三款最新的手表都搭载了同样的S10芯片，尺寸的大小决定了电池的大小，因此Ultra拥有最好的电池续航，其次是Series数字版跟SE。
